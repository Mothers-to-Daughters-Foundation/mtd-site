# QA Checklist

## Visual Parity

- [ ] Compare key pages with Wix site
- [ ] Spacing matches design
- [ ] Typography matches design
- [ ] Colors match brand palette
- [ ] Button styles consistent
- [ ] Card layouts match

## Functionality

- [ ] All forms submit correctly
- [ ] Zeffy embed works on `/donate`
- [ ] Navigation works everywhere
- [ ] Mobile responsive (test on multiple devices)
- [ ] Links work correctly
- [ ] Images load properly

## SEO

- [ ] All redirects working (test sample URLs)
- [ ] Sitemap generates correctly (`/sitemap.xml`)
- [ ] Metadata on all pages
- [ ] Canonical URLs correct
- [ ] Open Graph tags present
- [ ] Robots.txt configured

## Analytics

- [ ] GA4 realtime reporting
- [ ] Clarity sessions recording
- [ ] Route change tracking works
- [ ] Pageviews fire correctly

## Performance

- [ ] Lighthouse audit (90+ target)
- [ ] Image optimization
- [ ] Bundle size check
- [ ] First Contentful Paint < 1.8s
- [ ] Time to Interactive < 3.8s

## Accessibility

- [ ] Keyboard navigation works
- [ ] Focus indicators visible
- [ ] Screen reader test
- [ ] Color contrast (WCAG AA)
- [ ] Alt text on images
- [ ] Form labels associated
- [ ] ARIA labels where needed
